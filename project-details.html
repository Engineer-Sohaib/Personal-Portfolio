Project Overview

Al Tahaluf's is a full enterprise web platform built with Angular on the frontend and .NET Core on the backend. The system features a custom admin panel for user and content management, reusable Angular component architecture, lazy loading and RxJS-based state management — giving the business full control over their operations through a clean, responsive dashboard.

Secure admin login with role-based access control
RESTful API integration with Entity Framework & LINQ
Reusable Angular components with lazy loading
SQL Server database with optimised queries
Fully responsive design for all screen sizes
